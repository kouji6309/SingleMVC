# Single MVC

Single MVC 是一個方便使用、功能完整、單一檔案的 PHP 框架。它提供了以下功能：

* 多種目錄結構
* 基礎 MVC
* 路由覆寫
* 正確處理各種 HTTP 請求方法的資料
* 統一的方法取得輸入資料
* RESTful 方法對應
* 多國語系
* Model 自動載入
* 資料庫存取
* CURL 請求 (可多執行序)
* 效能紀錄與除錯
* JWT 編碼與解碼
* 檢查框架更新

更多功能與函數請見 [Wiki](https://github.com/kouji6309/SingleMVC/wiki)
