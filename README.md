# SingleMVC

SingleMVC 是一個方便使用、功能完整、單一檔案的 PHP 框架。它提供了以下功能：

* 多種目錄結構
* 基礎 MVC
* 自動載入與 Composer 安裝
* 路由覆寫
* RESTful 風格
* PHPUnit 測試
* 統一的輸入、輸出函數
* 簡易 SESSION、COOKIE 函數
* 多國語系函數
* 資料庫存取函數
* CURL 請求函數
* 效能紀錄與除錯函數
* JWT 編碼與解碼函數
* 檢查框架更新函數

更多功能與函數請見 [Wiki](https://github.com/kouji6309/SingleMVC/wiki)
